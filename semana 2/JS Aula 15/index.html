Comparação de textos

Dentro do javascript, podemos usar duas variaveis com nomes diferente e usar a tag ('==') para saber se uma delas tem o valor igual ao declarado dentro após o sinal de ('==')


Mesmo que a variável esteja com um tipo number e eu escrever uma String com ("") o javascript tem uma tipagem fraca, e ele vai entender após o comando de confimação ('==') que a string colocada é do tipo number*


Para confirmação, clicar com o botão direito e ir em inspecionar -> console. Lá vai ter as informações de "false" ou "True"
